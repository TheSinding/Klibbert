# Klibbert v2.0 alpha
A screenshot uploader

Depends on: **WILL CHANGE**
+ [Maim](https://github.com/naelstrof/maim)
+ [Slop](https://github.com/naelstrof/slop)
+ curl
+ sed
+ hash
+ xdg-open
